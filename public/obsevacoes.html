Informativos - Julho

Relatório de Incidentes e Manutenção - Servidores IXC e OpaSuite

Este relatório detalha os incidentes recentes e as ações tomadas para normalizar os serviços dos sistemas IXC e OpaSuite, culminando nos próximos passos para a migração para um novo servidor.

Histórico dos Incidentes

14 de Julho:

O servidor principal apresentou uma parada de operação, afetando diretamente os sistemas IXC e OpaSuite, bem como as aplicações Power. Foi necessário realizar um reinício forçado para restabelecer os serviços.

16 de Julho:

Foi identificada uma nova falha, ainda sob investigação, que resultou na reinicialização e travamento do servidor principal. Este incidente impactou diretamente as máquinas virtuais que hospedam os serviços do IXC e OpaSuite, exigindo um desligamento forçado do servidor para a normalização das operações.

17 de Julho:

Uma manutenção preventiva foi realizada no servidor que hospeda os sistemas IXC e OpaSuite. Após os ajustes necessários, o OpaSuite voltou a operar normalmente.

Próximos Passos – Migração para Novo Servidor

Com o objetivo de evitar futuras interrupções e melhorar a estabilidade dos serviços, um novo servidor já foi preparado para hospedar ambos os sistemas, IXC e OpaSuite. A próxima etapa crucial será a restauração do backup do dia 17 de julho neste novo ambiente.

Cronograma de Funcionamento - Sexta-feira, 18 de Julho

Para viabilizar a migração, o funcionamento dos sistemas na sexta-feira, 18 de julho, será o seguinte:

  • IXC: O sistema IXC atual estará disponível apenas para consulta. É crucial destacar que qualquer dado inserido nesse período será perdido, pois a base de dados em uso não será a migrada. A migração do IXC está programada para iniciar às 08h20, com previsão de término às 16h, devido ao volume do banco de dados.
  • OpaSuite: O sistema OpaSuite também será migrado no mesmo dia. A janela de migração estimada é entre 08h20 e 11h. As mesmas observações aplicadas ao IXC em relação à perda de dados inseridos no sistema antigo durante este período também se aplicam ao OpaSuite.

Este planejamento visa garantir uma transição suave para o novo ambiente, minimizando os impactos nas operações diárias.
